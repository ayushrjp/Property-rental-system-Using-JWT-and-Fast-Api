#  Property Rental Management System

A full-stack Property Rental Management System built using **FastAPI**, **JWT Authentication**, and **Supabase (PostgreSQL)**. This system helps manage properties, tenants, rental agreements, and payment tracking through a centralized dashboard.

---

##  Features

- 🔐 JWT Authentication (Secure Login)
- 🏠 Property Management (Add, View, Delete)
- 👤 Tenant Management
- 📄 Rental Agreements Handling
- 💳 Payment Tracking (Paid / Pending)
- 📊 Dashboard with insights
- ⚡ RESTful API using FastAPI

---

##  Tech Stack

- **Backend:** FastAPI (Python)
- **Database:** Supabase (PostgreSQL)
- **Authentication:** JWT (JSON Web Token)
- **Frontend:** HTML, CSS (Custom Dashboard UI)
- **Architecture:** Modular (API, Services, Schemas)
